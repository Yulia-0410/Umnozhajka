Multiplication Trainer with smart notifications. A desktop app built with Python and CustomTkinter that helps learn multiplication tables through periodic reminders.

Features:
- Customizable range of integers (including negative numbers)
- Adjustable answer time limit (5-30 seconds)
- Settings saved to JSON file

Perfect for schoolchildren to practice multiplication tables.

Тренажёр таблицы умножения с умными уведомлениями. Десктопное приложение на Python и CustomTkinter, которое помогает учить таблицу умножения с помощью периодических напоминаний.

Возможности:
- Настраиваемый диапазон целых чисел (включая отрицательные)
- Регулируемое время на ответ (5-30 секунд)
- Сохранение настроек в JSON

Идеально подходит для школьников, чтобы практиковать таблицу умножения.
